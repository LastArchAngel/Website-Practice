Location of script tags

Hmm, the location of our scripts DOES matter. It is better to place the scripts at the bottom of the body because these script tag compilation(they are being compiled) slows down the display portion of the website. Ooh, we can also place them in seperate files and import them :x How does this affect the load speed of the website?

If we use the script tag and use an external .js file(it can hold our function) then it will just behave as if the javascript code was written there. Remember that img tag doesn't have a closing bracket /img.

Some of the advantages of using external Javascript files is that it makes our code clearner, easier to read, and it improves load times.
